Corsicana Independent School District

New website for the Corsicana ISD

Some of the technologies/features of the site include:

  • All of the features and technologies of MC History
  • Built with WAI-ARIA accessibility standards in mind
  • Custom iCalendar API built with Biweekly(Java) and FullCalendar(JavaScript)
  • Built with intention of using Spring framework and Thymeleaf templating engine to provide admins with ability to create/edit/delete entire pages and customize their layouts and contents with a wide array of templates and template fragments
  • This website is still under construction. It's a large and complex project that will take time to complete.
