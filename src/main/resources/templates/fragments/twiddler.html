Twiddler

A single-page Twitter mock-up

There are two versions of the mock-up:

  1. Static JavaScript (Original, Dec 2017)
    • Mock data generator used to simulate live twiddles
    • Ability to toggle live feed
    • Ability to create a username and start twiddling
    • Ability to filter twiddles by hash tag or user
  2. React, Spring Boot, MySQL (Rebuild, Dec 2019)
    • React front-end | Mobile responsive | Infinite scrolling
    • Spring Boot REST API with Spring Security
    • Data persistence with MySQL, Hibernate
