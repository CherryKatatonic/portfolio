Twiddler

A simple Twitter mock-up made with Javascript

Some of the features of the mock-up include:

  • Random twiddle generator to simulate live twiddles
  • Ability to toggle live feed
  • Ability to create a username and start twiddling
  • Ability to filter twiddles by hash tag or user
  • This mock-up was not built with responsive web design principles. It is not yet mobile-friendly.
